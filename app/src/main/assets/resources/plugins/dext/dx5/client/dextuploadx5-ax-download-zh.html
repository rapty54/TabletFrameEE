※ 安装产品后，连续访问下载页面时，检验事项如下：
1。对于IE 10/11用户，请从兼容性视图列表中删除该站点（如果已设置）。
2. 进入信任的网站填加页面后，可以重新连接网站。
3. 在其他浏览器上安装文件时，下载文件后，把它移动到下载位置
4. 点击鼠标右键安装文件>以管理员身份运行。
5. 如上述的安装工作后，还发生同样的症状的话，请参考技术文件页面。
    -> 转到有技术文件的页面（必须连接到Internet。）
6. 由于其他的事项，不可完成安装时，请与管理员联系。

※ 运行安装文件时，出现拒绝访问错误的话：

2. 选择安装文件后，点击鼠标右键选择[属性]
3. 如果在【一般】标记上设定了‘阻止’的话，请点击【阻止解除】 。
4. 运行安装文件
