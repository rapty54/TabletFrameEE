※ Checking list when accessing the installation page continuously after the installation of the program on Windows.
  1. Run the 'taskschd.msc' to open the Task Scheduler.
  2. Check to see if there is the 'DEXTUploadX5-HDD' task and its status is 'running'.

HD application requires admin privileges to run, the user's account must have admin privileges

If your account does not have admin privileges, your account must be included in the 'Administrators' group.

For variouse reason if your account cannot be included in the 'Administrators' group, you need to go to the HD application installation location and manually run(double-click) the 'DEXTUploadX5-HDD.exe' program. (Requires elevation of privilege via UAC.)

※ HD application installation location in Windows
  • C:\Program Files\DEXTSolution\DEXTUploadX5 HD x64
  • C:\Program Files (x86)\DEXTSolution\DEXTUploadX5 HD
※ The file access was denied when executing the client program on Windows.
  1. Move to the folder where you downloaded file.
  2. Right-click the file and open its 'Property'.
  3. In the 'General' tab, unblock the protection of the file and click the 'Apply' button.
  4. Execute installation file.
※ How to install the HD application on macOS
  1. Download the installation package file("dextuploadx5-hd-mac.pkg"). (Note the browser pop-up blocker.)
  2. Run Finder and go to the location where the package is downloaded.
  3. Select the "dextuploadx5-hd-mac.pkg" file, right-click, and click the "Open" button to install.
※ How to uninstall the HD application on macOS
  1. Open the Spotlight search and type "Terminal" to run the terminal.
  2. Go to "/Users/CurrentUser/Applications/DEXTUploadX5-HD" folder by executing "cd ~/Applications/DEXTUploadX5-HD (enter)" in the terminal.
  3. Run "sh dextuploadx5-hd-uninstall.sh (Enter)" to delete the HD program.
