DEXTUploadX5 HD 설치 프로그램 다운로드

다운로드 (Windows용)

다운로드 (Windows용 64bit 프로그램)

다운로드 (macOS 10.13.6 이상, Safari 12 이상)

※ Windows에서 프로그램 설치 후 계속적으로 다운로드 페이지 접근 시 점검 사항
  1. 실행(Window 키 + R) 창을 열어 'taskschd.msc' 입력 후 실행
  2. 작업 스케줄러 루트 위치에서 'DEXTUploadX5-HDD' 작업이 존재하고, 상태가 '실행 중'인지 확인

HD 응용프로그램이 실행되기 위해서는 관리자 권한이 필요하기 때문에 사용자의 로그인 계정은 관리자 권한을 가지고 있어야 합니다.

사용자 계정이 관리자 권한을 가지고 있지 않다면, 해당 계정을 Administrators 그룹에 포함시켜야 합니다.

여러 가지 이유로 로그인 계정이 관리자 그룹에 포함될 수 없다면, HD 응용프로그램 설치 위치로 이동하여, 'DEXTUploadX5-HDD.exe' 프로그램을 수동으로 실행(더블 클릭)시켜야 합니다. (UAC 통한 권한 상승이 필요합니다.)

※ Windows에서 HD 응용프로그램 설치 위치
  • C:\Program Files\DEXTSolution\DEXTUploadX5 HD x64
  • C:\Program Files (x86)\DEXTSolution\DEXTUploadX5 HD
※ Windows에서 설치 파일 실행 시 액세스 거부 오류가 발생하는 경우
  1. 설치파일 다운로드 위치로 이동
  2. 설치파일을 선택 후 마우스 오른쪽 버튼을 클릭하여 '속성' 선택
  3. '일반' 탭에서 차단 설정이 되어있다면, '차단 해제' 버튼 클릭 후 적용
  4. 설치 파일 실행
※ macOS에서 프로그램 설치 방법
  1. 설치 패키지 파일('dextuploadx5-hd-mac.pkg')을 다운로드 (브라우저 팝업 차단 주의)
  2. 파인더(Finder)를 열고 다운로드 위치로 이동
  3. 'dextuploadx5-hd-mac.pkg' 파일을 선택한 다음, 오른쪽 클릭을 수행한 후, '열기' 버튼 클릭하여 설치 진행
※ macOS에서 프로그램 삭제 방법
  1. Spotlight 검색을 열어서 '터미널' 또는 'Terminal'을 입력하여 터미널을 실행
  2. 'cd ~/Applications/DEXTUploadX5-HD (엔터)'를 실행하여 '/Users/사용자/Applications/DEXTUploadX5-HD' 폴더로 이동
  3. 'sh dextuploadx5-hd-uninstall.sh (엔터)'를 실행하여 프로그램 삭제
